# SteamInfo

Модуль, который дополняет профиль игроков привязавших свой Steam информацией о Steam ID в разных форматах.

Module that complements the profile of players who have linked their Steam information in different formats.

## Installation

Download the latest release and install it via the Flute CMS admin panel.

Current version: **1.0.1**
